JavaScript exercises

  1. Run npm install
  2. The examples are divded into three sections
    1. array.js
    2. math.js
    3. string.js
  3. Follow the comments in each section to guide you along.
    Reading the test files will also help, try not to look helper.js becuse they may contain solutions(or at least one way to do them)
  4. To run test, type this in terminal npm test spec/{{section}}Spec.js
    For an example to test the array methods: npm test spec/arraySpec.js
  5. Get all the test to pass
  6. There are many ways to get the test to pass, be creative!
